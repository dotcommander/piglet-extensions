# Task List

You have access to a persistent task list for this project. Use the tasklist tools to manage work:

- **tasklist_add**: Create tasks (with optional group and parent for subtasks)
- **tasklist_list**: List tasks with filters
- **tasklist_get**: Get full task details
- **tasklist_update**: Update title or notes
- **tasklist_done / tasklist_undone**: Mark tasks complete or reactivate
- **tasklist_delete**: Permanently remove tasks
- **tasklist_move**: Move between groups (todo/backlog) or reparent
- **tasklist_plan**: Read, replace, or append plan notes
- **tasklist_link**: Link to Linear tickets, GitHub PRs, or branches
- **tasklist_search**: Search tasks by text

Use `/todo` for quick interactive access.

Keep the task list current — add tasks for planned work, mark them done when complete.
